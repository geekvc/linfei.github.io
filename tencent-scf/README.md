之前尝试用 Cloudflare Workers 做反向代理，但国内访问 `*.workers.dev` 域名经常出现连接失败（DNS 解析被干扰，是众所周知的老问题）。

既然你已经在腾讯云买了 `linfei.wang` 域名，我们改用 **腾讯云函数（SCF）+ API 网关**，同一账号体系，国内节点访问稳定，免费额度也很充足，10 分钟内可以搞定。

## 第一步：开通云函数 SCF

1. 打开 https://console.cloud.tencent.com/scf ，用你腾讯云账号登录（跟买域名的是同一个账号）
2. 如果是第一次使用，会提示"开通服务"，直接同意开通即可（免费，不产生费用）

## 第二步：新建函数

1. 点击「**新建**」
2. 选择「**自定义创建**」
3. 关键配置：
   - 函数名称：`linfei-ai-chat`
   - 地域：选择离你近的，比如 **广州** 或 **上海**（后面记得 API 网关也选同一个地域）
   - 运行环境：**Node.js 18.15** 或更高版本
   - 提交方式：**在线编辑**
   - 模板：选择「**空白函数**」/ Hello World 模板即可
   - 执行方法：保持默认 `index.main_handler`
4. 点击「完成」创建

## 第三步：替换函数代码

1. 创建完成后，会自动进入函数详情页的「函数代码」标签（或点击进去）
2. 左侧文件列表找到 `index.js`，点击打开
3. **全选删除**里面的默认代码
4. 打开你本地文件 `/Users/tywang/workspace/linfeiwang/tencent-scf/index.js`，**全选复制全部内容**
5. 粘贴到腾讯云在线编辑器里
6. 点击上方「**保存并安装依赖**」或「**部署**」按钮（因为代码只用了 Node.js 内置的 `https` 模块，没有额外依赖，很快就能部署完成）

## 第四步：配置环境变量（安全存放 API Key）

1. 在函数详情页，找到「**函数配置**」标签
2. 找到「环境变量」区域，点击「编辑」
3. 添加一个环境变量：
   - 变量名称：`ZHIPU_API_KEY`
   - 变量值：`6bb70c8381784fddafcf881bf32a20a9.ZaeVDGjdHZTqEnck`
4. 保存

## 第五步：创建触发器（让它能通过网址访问）

1. 在函数详情页，找到「**触发管理**」标签
2. 点击「创建触发器」
3. 触发方式选择：**API 网关触发器**
4. 关键配置：
   - API 名称：随意，比如 `linfei-ai-chat-api`
   - 鉴权方式：**免鉴权**（因为要给浏览器直接调用）
   - 请求方法：**ANY** 或者勾选 POST + OPTIONS（两者都要支持，OPTIONS 是浏览器 CORS 预检请求）
   - 是否启用响应集成/集成响应：如果有这个选项，选择「**是**」（这样函数返回的 statusCode/headers 才能生效，否则跨域会失败）
5. 提交后，会生成一个访问路径，形如：

```
https://service-xxxxxxxx-1234567890.gz.apigw.tencentcs.com/release/linfei-ai-chat
```

把这个完整地址复制下来发给我。

## 第六步（可选，但强烈建议）：绑定自定义域名，避免国内访问异常

腾讯云 API 网关默认域名（`xxx.apigw.tencentcs.com`）国内访问通常是稳定的，可以先直接用这个测试。如果后续想要更美观或者更稳妥，可以在 API 网关控制台给这个 API 绑定一个子域名，比如 `chat.linfei.wang`（走你自己腾讯云域名的解析，同账号体系操作会非常顺畅，不像 Cloudflare 那样容易卡住）。这一步不着急，先用默认域名跑通功能。

## 常见问题

- **调用报错 CORS**：检查 API 网关触发器是否开启了"响应集成"，以及函数返回内容里 `headers` 字段是否正确回传
- **函数报错找不到环境变量**：检查函数配置里 `ZHIPU_API_KEY` 是否保存成功，保存后需要重新发布函数版本才生效
- **想测试函数是否正常**：可以直接在函数详情页的「**测试**」标签，构造一个测试事件：
  ```json
  {
    "httpMethod": "POST",
    "headers": { "origin": "https://linfei.wang" },
    "body": "{\"messages\":[{\"role\":\"user\",\"content\":\"你好，介绍一下你自己\"}]}"
  }
  ```
  点击测试，看返回结果里 `body` 字段是否包含正常的 AI 回复内容
